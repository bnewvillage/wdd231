MotorCity UAE - Project Site Plan

Site Name

MotorCity UAE

This name represents the Motorcycle Accessory shop specifically the branch in the Unite Arab Emirates.

Site Purpose

The MotorCity UAE website is designed for the internal team of a motorcycle shop. It will include:

  • A home page with highlights relevant to employees.
  • A warranty request page.
  • An order request page.

Scenarios

1. Warranty Application

What information do I need to request for a warranty application for products from the brand Klim?

2. Special Order Request

How do I request for a special order to be placed?

Color Schema

  • Primary Color: #000957 (Background Color)
  • Secondary Color: #344CB7 (Header and Footer Color)
  • Accent Color 1: #577BC1 (Button colors and hover effects)
  • Accent Color 2: #FFEB00 (Highlight effects)

Typography

  • Headings: Roboto 700
  • Subheadings: Roboto 400
  • Content Text: Roboto 300, Roboto 100

Wireframe Considerations

The site will be designed for both desktop and mobile views, ensuring responsiveness.

Evaluation & Best Practices

This project will be assessed using common tools for:

  • HTML validation
  • Accessibility compliance
  • Best practices
  • Color contrast testing
  • SEO optimization
  • Performance evaluation
